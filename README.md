# ClarifiED Announcements

An AI-powered tool to transform educational announcements into engaging, HTML-formatted messages for D2L Brightspace.

## Tech Stack
- **React 19**
- **Vite**
- **Tailwind CSS**
- **Google Gemini 3 Pro API**

## Local Development

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set your API Key in a `.env` file:
   ```env
   VITE_API_KEY=your_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment
This app is designed to be deployed to Netlify or Vercel. Ensure you add your `API_KEY` to the environment variables in your hosting provider's dashboard.
